# 📊 Sales Data Analysis Project

## 📌 Overview
This project analyzes retail sales data using SQL (BigQuery) and Power BI to generate insights on sales performance, customers, products, and payment methods.

---

## 🧹 Data Cleaning
The dataset was cleaned using BigQuery to ensure data quality before analysis.

### Key steps:
- Removed duplicate records  
- Filtered numeric StockCode values  
- Handled missing values  
- Removed outliers in age  
- Applied TRIM function to clean text fields (e.g., removing extra spaces)  
- Sampled the dataset due to large data size to improve performance and processing efficiency  

SQL queries are available in the `data_cleaning.sql` file.
